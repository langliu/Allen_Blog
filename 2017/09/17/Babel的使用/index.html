babel-node

我们运行js的时候使用的命令是 node <filename> ，如果你的js代码中使用了nodejs暂时还未支持的语法，则代码会报错。这时，Babel提供给我们了一个 babel-node 命令，你可使用 babel-node 命令替换 node 命令直接运行ES2015的代码，而不需要将其编译成ES5的语法后运行编译后的代码。
