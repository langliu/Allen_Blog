Numpy中的数组

2018-07-29
  • Python

NumPy 中定义的最重要的对象是称为 ndarray 的N维数组类型。它描述相同类型的元素集合。可以使用基于零的索引访问集合中的项目。

ndarray 中的每个元素在内存中使用相同大小的块。 ndarray中的每个元素是数据类型对象的对象(称为 dtype)。

从 ndarray 对象提取的任何元素(通过切片)由一个数组标量类型的 Python 对象表示。
